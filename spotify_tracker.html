Spotify Tracker


Project Proposal

Show and Tell 1

About

Making use of spotify Api Going to track user statistics. Being able to login securely using spotify authentication. Saving spotify user data and statistics locally to a Room database. Such as: Listening time, Recently played, Favorite genre, Favorite artists. Music will be suggested based on local weather, songs that are trending, or the user can enter their current mood to find songs that fit the mood/vibe.
